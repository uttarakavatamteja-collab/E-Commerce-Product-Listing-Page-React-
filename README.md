# Kalki Store - E-Commerce Product Listing Page

A premium, production-ready E-commerce Product Listing Page built with React and Tailwind CSS. This application demonstrates modern UI/UX practices including glassmorphism, responsive design, and real-time data integration.

## ✨ Features

- **Product Grid**: Fully responsive layout (4 columns desktop, 2 tablets, 1 mobile).
- **Category Filtering**: Seamlessly filter products by categories (Electronics, Jewelry, etc.).
- **Real-time Search**: Instant search functionality that works in combination with category filters.
- **Product Details Modal**: Deep-dive into product details with an accessible, animated modal.
- **Loading & Error Handling**: Polished feedback states with retry logic.
- **Premium Design**: Built with a sleek Glassmorphism theme and smooth animations.

## 🚀 Tech Stack

- **Framework**: [React.js](https://reactjs.org/) (Hooks used: `useState`, `useEffect`, `useMemo`, `useCallback`)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **API**: [FakeStoreAPI](https://fakestoreapi.com/)

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/uttarakavatamteja-collab/E-Commerce-Product-Listing-Page-React-.git
   cd E-Commerce-Product-Listing-Page-React-
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📱 Responsive Breakpoints

- **Desktop (>= 1024px)**: 4 columns
- **Tablet (>= 640px)**: 2 columns
- **Mobile (< 640px)**: 1 column

## 🎨 UI/UX Design

The application uses a "Glassmorphism" aesthetic with:
- `backdrop-filter: blur(10px)`
- HSL-tailored semi-transparent background colors
- Subtle border glows and hover transitions
- Accessible focus trapping in modals
